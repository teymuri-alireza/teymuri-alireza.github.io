# Alireza Teymuri

I've created a interactive terminal interface with javascript for this repo.

check it out: [teymuri-alireza.github.io](https://teymuri-alireza.github.io/)

## supported commands

- help
- sudo
- ls
- whoami
- pwd
- man
- echo
- date
- clear
