# Alireza Teymuri

Just an interactive terminal interface in javascript.

check it out: [teymuri-alireza.github.io](https://teymuri-alireza.github.io/)

## Supported Commands

- help
- sudo
- ls
- whoami
- pwd
- cd
- man
- echo
- date
- clear
- ...
